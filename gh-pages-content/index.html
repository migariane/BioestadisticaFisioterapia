Objetivos del Curso

  • Leer críticamente la literatura científica en fisioterapia
  • Analizar datos clínicos con herramientas estadísticas apropiadas
  • Interpretar correctamente p-valores, intervalos de confianza y tamaños del efecto
  • Elegir la prueba adecuada según el tipo de variable y diseño del estudio
  • Reproducir análisis usando R y el paquete BioEstatR
  • Comunicar resultados estadísticos de forma clara y rigurosa

Contenido del Curso

Nota

Cada tema incluye: fundamentos teóricos con desarrollo matemático, aplicación clínica en fisioterapia, ejemplos numéricos paso a paso, y práctica guiada con R + BioEstatR.

Instalación de BioEstatR

# Instalar BioEstatR desde GitHub
install.packages("remotes")
remotes::install_github(
  "migariane/BioEstatR"
)

# Cargar el paquete
library(BioEstatR)

# Verificar instalación
packageVersion("BioEstatR")

Tip

Funciones principales de BioEstatR

Función Uso
freq() Estadística descriptiva
icm() IC para la media
icp() IC para proporción
testt() Test medias t (1 y 2 muestras)
testp() Test proporciones (1 y 2 muestras)
tabla2x2() Tablas 2×2
rls() Regresión lineal
nm() / np() Tamaño muestral
